Принципы

Самое важное.

Например

Никогда не использовать чисто черный #000.

Никогда не использовать fade без изменения положения.

Никогда не использовать одинаковые анимации дважды подряд.

Каждая секция должна иметь собственную визуальную идентичность.

Каждый hover должен ощущаться физически.

Каждая карточка должна иметь глубину.

Между секциями должно быть много воздуха.
